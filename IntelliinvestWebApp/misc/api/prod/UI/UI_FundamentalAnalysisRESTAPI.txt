Fundamental Analysis
--------------------

1. For a given stock get latest fundamental analysis
http://104.199.173.131/IntelliinvestWebApp/rest/stock/getLatestFundamentalAnalysis?id=SBIN

2. For a given stock get fundamental analysis for date
http://104.199.173.131/IntelliinvestWebApp/rest/stock/getFundamentalAnalysisForDate?id=SBIN&date=2016-10-19

3. For a given stock and attribute, return the time series for stock(yearQuarter, attributeValue) 
and industry average(date, attributeValue).

http://104.199.173.131/IntelliinvestWebApp/rest/stock/getFundamentalAnalysisTimeSeries?id=SBIN&attrName=alMarketCap

4. For a given industry get latest industry analysis
http://104.199.173.131/IntelliinvestWebApp/rest/industry/getLatestIndustryFundamentals?name=BANKING

5. For a given industry and date, get industry analysis
http://104.199.173.131/IntelliinvestWebApp/rest/industry/getIndustryFundamentalsForDate?name=BANKING&date=2016-10-19

6. For a given industry and attribute, return the time series for industry(date, attributeValue)

http://104.199.173.131/IntelliinvestWebApp/rest/industry/getIndustryAnalysisTimeSeries?name=BANKING&attrName=alMarketCap

7. For a given industry get top 10 stocks
http://104.199.173.131/IntelliinvestWebApp/rest/stock/getTopStocksForIndustry?name=BANKING
